5.3.4 Caso Práctico

En esta práctica, desarrollaremos una aplicación Android en Java que permitirá gestionar reseñas de películas y series. La aplicación estará diseñada utilizando una arquitectura Single Activity y hará uso de las siguientes herramientas y conceptos clave:

  • RecyclerView: Para mostrar una lista de recetas almacenadas.
  • Room: Para almacenar los datos de las recetas en una base de datos local.
  • Arquitectura MVVM: Para separar responsabilidades y facilita el mantenimiento y escalabilidad de la aplicación.
  • Gestos (Swipe): Para eliminar elementos del RecyclerView deslizando hacia la izquierda.
  • Galería: Para seleccionar una imagen asociada a una receta.
