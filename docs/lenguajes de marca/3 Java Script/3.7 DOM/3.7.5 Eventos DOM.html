3.7.5 Eventos DOM

Los eventos en el DOM son acciones o sucesos que ocurren en la página web, como un clic del usuario, el envío de un formulario, o incluso el paso del mouse sobre un elemento. Estos eventos pueden ser escuchados y manejados para realizar interacciones dinámicas.

1. Modelo de Eventos del DOM​

El modelo de eventos describe cómo se propagan los eventos a través del árbol DOM. Hay dos fases principales:

Burbujeo (Bubbling)​

  • El evento se genera en el nodo donde ocurrió (elemento objetivo) y "burbujea" hacia arriba en el árbol DOM, llegando hasta el nodo document.
  • Es la fase por defecto al añadir eventos sin especificar una fase.

Ejemplo de flujo:

<div id="contenedor">
  <button id="boton">Clic aquí</button>
</div>
<script>
  const contenedor = document.getElementById('contenedor');
  const boton = document.getElementById('boton');

  // Evento en el contenedor
  contenedor.addEventListener('click', () => {
    console.log('Evento en contenedor (burbujeo)');
  });

  // Evento en el botón
  boton.addEventListener('click', () => {
    console.log('Evento en botón');
  });
</script>

Salida esperada al hacer clic en el botón:

  1. Se ejecuta el evento del botón.
  2. El evento "burbujea" y se ejecuta el del contenedor.

Captura (Capturing)​

  • El evento se propaga desde el nodo raíz (document) hacia abajo hasta el nodo objetivo.
  • Para usar esta fase, se pasa true como tercer parámetro en addEventListener().

Ejemplo de flujo:

contenedor.addEventListener(
  'click',
  () => {
    console.log('Evento en contenedor (captura)');
  },
  true // Indica que el evento se maneja en la fase de captura
);

2. Manejo de Eventos​

addEventListener()​

  • Descripción:

    • Asocia una función manejadora de eventos a un elemento.
    • Permite especificar si se maneja en la fase de burbujeo o captura.

  • Sintaxis:

elemento.addEventListener(evento, manejador, usoDeCaptura);
  • Ejemplo:
const boton = document.getElementById('boton');
boton.addEventListener('click', () => {
  console.log('Botón clickeado');
});

removeEventListener()​

  • Descripción:

    • Elimina un manejador previamente añadido con addEventListener().

  • Nota:

    • La función manejadora debe ser una referencia para que pueda ser eliminada.

  • Ejemplo:

function manejarClic() {
  console.log('Evento activado');
}

boton.addEventListener('click', manejarClic);
boton.removeEventListener('click', manejarClic);

3. Tipos de Eventos Comunes​

Eventos del Mouse​

EventoDescripción
clickSe dispara cuando el usuario hace clic.
dblclickSe dispara con un doble clic.
mouseoverSe dispara cuando el mouse pasa por encima.
mouseoutSe dispara cuando el mouse sale del elemento.
contextmenuSe dispara al hacer clic derecho.

Eventos del Teclado​

EventoDescripción
keydownSe dispara cuando una tecla es presionada.
keyupSe dispara al soltar una tecla.
keypress(obsoleto) Similar a keydown, pero menos usado.

Eventos de Formularios​

EventoDescripción
submitSe dispara al enviar un formulario.
changeSe dispara al cambiar el valor de un campo.
inputSe dispara cuando se escribe en un campo.
focusSe dispara cuando un campo gana el foco.

4. Propiedades y Métodos del Objeto Evento​

Cuando ocurre un evento, se genera un objeto de tipo Event que contiene información sobre el evento y permite manipularlo.

target​

  • Descripción:

    • Devuelve el elemento que disparó el evento.

  • Ejemplo:

document.addEventListener('click', (evento) => {
  console.log(evento.target); // Muestra el elemento clickeado
});

preventDefault()​

  • Descripción:

    • Evita la acción predeterminada asociada al evento.

  • Ejemplo:

const enlace = document.getElementById('enlace');
enlace.addEventListener('click', (evento) => {
  evento.preventDefault(); // Evita que se abra el enlace
  console.log('Acción predeterminada evitada');
});

stopPropagation()​

  • Descripción:

    • Detiene la propagación del evento en el árbol DOM.

  • Ejemplo:

const contenedor = document.getElementById('contenedor');
const boton = document.getElementById('boton');

contenedor.addEventListener('click', () => {
  console.log('Evento en contenedor');
});

boton.addEventListener('click', (evento) => {
  evento.stopPropagation(); // Detiene la propagación
  console.log('Evento en botón, propagación detenida');
});

Salida esperada al hacer clic en el botón:

  • Solo se ejecuta el evento del botón.

Ejemplo Completo​

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Manejo de Eventos</title>
</head>
<body>
  <div id="contenedor">
    <button id="boton">Hacer clic</button>
    <a href="https://example.com" id="enlace">Ir a Example</a>
  </div>

  <script>
    const contenedor = document.getElementById('contenedor');
    const boton = document.getElementById('boton');
    const enlace = document.getElementById('enlace');

    // Evento de clic en el botón
    boton.addEventListener('click', (evento) => {
      console.log('Botón clickeado');
      evento.stopPropagation(); // Detener propagación
    });

    // Evento en el contenedor
    contenedor.addEventListener('click', () => {
      console.log('Contenedor clickeado');
    });

    // Evitar acción predeterminada del enlace
    enlace.addEventListener('click', (evento) => {
      evento.preventDefault();
      console.log('Navegación evitada');
    });
  </script>
</body>
</html>
