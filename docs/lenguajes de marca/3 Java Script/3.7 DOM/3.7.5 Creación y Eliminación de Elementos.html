3.7.5 Creación y Eliminación de Elementos

Manipular dinámicamente los elementos del DOM es una tarea esencial en el desarrollo web. La creación, modificación y eliminación de elementos nos permite construir interfaces dinámicas e interactivas sin recargar la página. En esta sección, exploraremos en detalle cómo crear, añadir, clonar y eliminar elementos del DOM. Los principales métodos para ello son...

MétodoDescripción
document.createElement()Crea un nuevo elemento HTML.
document.createTextNode()Crea un nodo de texto.
appendChild()Añade un nodo al final de otro nodo.
insertBefore()Inserta un nodo antes de otro existente.
cloneNode()Clona un nodo, con o sin contenido.
removeChild()Elimina un nodo hijo.
replaceChild()Reemplaza un nodo hijo con otro nodo.

Crear Elementos​

El DOM proporciona métodos como document.createElement() y document.createTextNode() para generar nuevos nodos que posteriormente podemos añadir al documento.

1. document.createElement()​

  • Descripción:

    • Crea un nuevo elemento HTML con el nombre de etiqueta especificado.
    • El elemento creado existe en memoria, pero no es visible en el DOM hasta que se añada explícitamente.

  • Sintaxis:

const elemento = document.createElement('nombreDeEtiqueta');
  • Ejemplo Básico:
const nuevoDiv = document.createElement('div');
nuevoDiv.textContent = "Este es un nuevo div";
  • Personalización del Elemento:
const boton = document.createElement('button');
boton.textContent = "Haz clic aquí";
boton.setAttribute('id', 'miBoton');
boton.classList.add('btn', 'btn-primario');
console.log(boton.outerHTML);
// <button id="miBoton" class="btn btn-primario">Haz clic aquí</button>

2 document.createTextNode()​

  • Descripción:

    • Crea un nodo de texto, ideal para añadir contenido textual a un elemento sin interpretarlo como HTML.

  • Sintaxis:

const texto = document.createTextNode('Texto simple');
  • Ejemplo:
const parrafo = document.createElement('p');
const texto = document.createTextNode('Este es un párrafo de texto.');
parrafo.appendChild(texto);
document.body.appendChild(parrafo);

Añadir Elementos al DOM​

Una vez creados los elementos, es necesario añadirlos al DOM. Para esto, podemos utilizar métodos como appendChild() e insertBefore().

1. appendChild()​

  • Descripción:

    • Añade un nodo al final de la lista de nodos hijos de un elemento.

  • Sintaxis:

parentNode.appendChild(nuevoNodo);
  • Ejemplo:
const lista = document.getElementById('miLista');
const nuevoElemento = document.createElement('li');
nuevoElemento.textContent = "Elemento 1";
lista.appendChild(nuevoElemento);

2. insertBefore()​

  • Descripción:

    • Inserta un nodo antes de otro nodo hijo existente en un elemento.

  • Sintaxis:

parentNode.insertBefore(nuevoNodo, nodoReferencia);
  • Ejemplo:
const lista = document.getElementById('miLista');
const nuevoElemento = document.createElement('li');
nuevoElemento.textContent = "Nuevo Elemento";

const primerElemento = lista.firstChild; // Nodo de referencia
lista.insertBefore(nuevoElemento, primerElemento);

Clonar Elementos​

El método cloneNode() permite copiar un nodo existente, con o sin sus nodos hijos.

cloneNode()​

  • Descripción:

    • Crea una copia exacta de un nodo.
    • Puede clonar solo la estructura (false) o la estructura y el contenido (true).

  • Sintaxis:

const clon = nodoOriginal.cloneNode(deep);
  • Ejemplo:
const original = document.getElementById('miDiv');
const clon = original.cloneNode(true); // Clona con el contenido
document.body.appendChild(clon);

Eliminar Elementos​

Eliminar elementos es igual de importante que crearlos, especialmente para mantener un DOM limpio y eficiente.

1. removeChild()​

  • Descripción:

    • Elimina un nodo hijo del DOM.
    • Requiere una referencia al nodo padre y al nodo hijo que se desea eliminar.

  • Sintaxis:

parentNode.removeChild(nodoHijo);
  • Ejemplo:
const lista = document.getElementById('miLista');
const primerElemento = lista.firstChild; // Nodo hijo
lista.removeChild(primerElemento);

2. replaceChild()​

  • Descripción:

    • Reemplaza un nodo hijo existente por otro nodo.

  • Sintaxis:

parentNode.replaceChild(nuevoNodo, nodoExistente);
  • Ejemplo:
const lista = document.getElementById('miLista');
const nuevoElemento = document.createElement('li');
nuevoElemento.textContent = "Elemento Reemplazado";

const viejoElemento = lista.firstChild; // Nodo a reemplazar
lista.replaceChild(nuevoElemento, viejoElemento);

Ejemplo​

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Manipulación del DOM</title>
</head>
<body>
  <div id="contenedor">
    <ul id="miLista">
      <li>Elemento Estático 1</li>
      <li>Elemento Estático 2</li>
    </ul>
    <button id="agregar">Agregar Elemento</button>
    <button id="reemplazar">Reemplazar Primer Elemento</button>
    <button id="eliminar">Eliminar Último Elemento</button>
  </div>

  <script>
    const lista = document.getElementById('miLista');
    const botonAgregar = document.getElementById('agregar');
    const botonReemplazar = document.getElementById('reemplazar');
    const botonEliminar = document.getElementById('eliminar');

    // Agregar un nuevo elemento
    botonAgregar.addEventListener('click', () => {
      const nuevoElemento = document.createElement('li');
      nuevoElemento.textContent = `Elemento Dinámico ${lista.childElementCount + 1}`;
      lista.appendChild(nuevoElemento);
    });

    // Reemplazar el primer elemento
    botonReemplazar.addEventListener('click', () => {
      if (lista.firstChild) {
        const nuevoElemento = document.createElement('li');
        nuevoElemento.textContent = "Elemento Reemplazado";
        lista.replaceChild(nuevoElemento, lista.firstChild);
      } else {
        alert('No hay elementos para reemplazar.');
      }
    });

    // Eliminar el último elemento
    botonEliminar.addEventListener('click', () => {
      if (lista.lastChild) {
        lista.removeChild(lista.lastChild);
      } else {
        alert('No hay elementos para eliminar.');
      }
    });
  </script>
</body>
</html>
